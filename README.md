# 矿工星火助手

矿工星火助手是一个为矿工游戏玩家设计的一站式辅助平台，提供多种功能模块帮助玩家更好地管理游戏资源和策略。

## 功能模块

目前已实现的功能模块：

1. **主页面** - 功能模块导航中心
2. **阵法图计算器** - 计算不同阵法组合的产出和成本

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- Font Awesome 图标库

## 如何使用

1. 打开 `index.html` 文件进入主页面
2. 点击各个功能模块图标进入相应功能页面
3. 在阵法图计算器页面，输入各个阵法的数量，系统会自动计算总产出、总成本和利润
